# NeuroRead AI 🧠

**Making any webpage readable for neurodivergent users through real-time AI multimodal transformation.**

> **Google Big Code Hackathon 2026 · Accessibility Track**  
> Powered by **FastAPI**, **Groq Cloud** (Llama 3.1 70B, Vision, Whisper), and **Redis**.

---

## Project Overview

NeuroRead AI is a Chrome extension + FastAPI backend that uses AI to make web content accessible for ADHD, Dyslexia, and Autism spectrum users.

## Tech Stack

- **Backend**: FastAPI, Python 3.10+
- **AI**: Groq Cloud (Llama 3.1, Vision, Whisper)
- **Caching**: Redis
- **Extension**: Chrome MV3
